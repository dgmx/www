## 2. Concepto de aplicación
- Programa informático diseñado como herramienta para realizar tareas
- Pertenecen al software de aplicación (contacto directo con usuario final)
- Orientadas a automatización de tareas complicadas
- Tipos
  - A medida: gran potencia, rapidez, problema específico
  - Paquetes integrados: menos adaptabilidad, abanico variado de aplicaciones
- Ejemplos
  - Procesadores de texto
  - Hojas de cálculo
  - Bases de datos
  - Presentaciones
  - Diseño gráfico
  - Cálculo
  - Finanzas
  - Compresión de archivos
  - Presupuestos
  - Gestión de empresas
## 3. Aplicaciones informáticas de propósito general
- Creadas para resolver problemas generales de trabajo
- Evitan costes elevados de programas específicos
### 3.1. Procesadores de texto
- Creación, edición, modificación y procesamiento de documentos con formato
- Funciones habituales
  - Elaboración, edición y formato de textos
  - Impresión con distintos formatos
  - Realización de gráficos
  - Diccionarios
  - Creación de formularios
  - Manejo de grandes documentos
  - Funciones aritméticas
- Correctores ortográficos y gramaticales
- Ejemplos: Microsoft Word, Office Word Online, LibreOffice Writer, Google Docs
### 3.2. Hojas de cálculo
- Manipulación de datos numéricos y alfanuméricos en tablas (filas y columnas)
- Celda: unidad básica (valores y fórmulas)
- Cálculos complejos con fórmulas/funciones
- Creación de gráficas
- Ejemplos: Microsoft Excel, Google Sheets, Apple Numbers, LibreOffice Calc
### 3.3. Sistemas gestores de bases de datos (SGBD)
- Base de datos: conjunto de datos persistentes, estructurados e interrelacionados
- SGBD: colección de programas para creación y mantenimiento de BD
- Funciones fundamentales
  - Definición de datos
  - Manipulación de datos
  - Control de datos
- Ejemplos: Microsoft Access, MongoDB, Hadoop, MySQL, PostgreSQL, Oracle CDM, LibreOffice Base
### 3.4. Comunicaciones
- Envío de mensajes e información por computadora
- Tipos
  - Correo electrónico
  - Correo de voz
  - Video
  - Mensajes de texto
  - Mensajería
  - Llamadas telefónicas
- Ejemplos: Internet Explorer, Microsoft Outlook, Mozilla Firefox, Google Chrome
### 3.5. Presentaciones
- Automatización de creación de ayudas visuales (charlas, conferencias)
- Combinan formas, textos, colores, movimientos y sonidos
- Ejemplos: Microsoft PowerPoint, Prezi, Google Slides, LibreOffice Impress
### 3.6. Diseño gráfico
- Retocar/modificar imágenes y realizar dibujos
- Operaciones
  - Añadir/quitar partes
  - Superponer imágenes
  - Retocar colores/líneas
  - Cambiar brillo/contraste/tamaño
  - Comprimir imagen
  - Cambiar formato
  - Aplicar filtros
  - Recortar
- Ejemplos: Adobe Photoshop, Corel Draw, Google Picasa, Gimp
### 3.7. Paquete integrado
- Herramienta compuesta por varias aplicaciones de uso común
- Aplicaciones interactúan e intercambian datos entre sí
- Agrupadas para dar servicio completo
- Ejemplos
  - LibreOffice (Writer, Calc, Impress, etc.)
  - Microsoft Office (Word, Excel, PowerPoint, etc.)
## 4. Aplicaciones informáticas para la gestión empresarial
- Sistemas integrados de gestión (ERP)
  - Cubren todas las áreas funcionales: finanzas, producción, compras, ventas, RRHH
  - Módulos adaptables a necesidades del cliente
  - Uso integrado vs aplicaciones aisladas por módulo
### 4.1. Gestión financiero-contable
- Automatización: asientos, extractos, libros diarios
- Seguridad, protección e integridad de la información
- Procesos básicos
  - Introducción y consultas de apuntes en libro diario
  - Informes de libro diario
  - Extractos de libro mayor
  - Balance de sumas y saldos
  - Ficheros de bancos (altas, bajas, modificaciones, consultas)
  - Previsión de pagos, cobros y riesgos
  - Gestión del IVA
### 4.2. Gestión de personal
- Gestión de recursos humanos
- Gestión de plantilla
  - Actualización/modificación de datos
  - Filiación completa (datos personales, médicos)
  - Historial laboral
  - Datos salarios e incentivos
  - Historial formativo
- Nóminas
  - Retribución bruta según datos contractuales e historial
  - Deducciones (Seguridad Social, IRPF, otros gastos)
  - Retribución neta
  - Procesamiento en lotes (batch)
### 4.3. Gestión comercial
- Gestión de pedidos y facturación
  - Esquemas de transacciones o trabajo en lotes
  - Ejecución interactiva para confirmaciones inmediatas
- Marketing
  - Apoyo en grandes bases de datos
  - Optimización de accesos habituales
- Análisis de mercado
  - Simulaciones complejas
  - Análisis estadísticos sofisticados
### 4.4. Control de stock y almacén
- Control de existencias almacenadas
- Nivel básico: actualización registro de inventario según ventas
- Nivel sofisticado
  - Predicción momento de aprovisionamiento (minimizar existencias)
  - Previsión ritmo producción según nivel ventas
- Unidos a automatización de producción: sistemas de planificación de recursos de fabricación
## 5. Aplicaciones en la nube
- Software y servicios ejecutados en internet (red de servidores)
- Requiere dispositivo con conexión a Internet
- Office 365
  - Integrada en la nube
  - Acceso en tiempo real desde cualquier dispositivo
  - Suite completa desde navegador
- OneDrive
  - Microsoft
  - Gratuita
  - Menos herramientas que Office 365
- Drive
  - Google
  - Almacenamiento de todo tipo de archivos
  - Creación/edición online con Google Docs
  - Edición colaborativa en tiempo real
- Dropbox
  - Centralización de contenido
  - Sincronización mediante disco duro virtual
  - Acceso remoto global
- Zyncro
  - Almacenamiento + comunicaciones para empresas/pymes
  - Mensajería colaborativa
  - Edición documentos entre varias personas
  - Gestor de tareas y reuniones
- Salesforce
  - Herramientas para ventas y marketing
  - Seguimiento de clientes, potenciales y contratos
  - Aumento de lealtad y satisfacción del cliente
## 6. Aplicación al contexto escolar y laboral
- Contexto escolar
  - ESO y bachillerato: TIC I de 1º de bachillerato
  - Ciclos formativos: DAM, DAW, SMR
- Contexto laboral
  - Entornos variados: banca, universidad, medicina, líneas aéreas
  - Tendencia: aplicaciones completamente online
## 7. Conclusión
- Importancia de aplicaciones de propósito general en trabajo diario
- Aplicaciones de gestión empresarial imprescindibles para eficiencia
- Buena selección de aplicaciones es clave
- Tendencia hacia aplicaciones en la nube y trabajo colaborativo
