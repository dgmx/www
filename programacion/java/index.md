# Programación en Java

Material didáctico sobre programación Java orientada a objetos, incluyendo manuales teóricos, ejercicios prácticos y ejemplos resueltos con diagramas UML.

## Manual

- [Manual de Programación Java. Diagramas de Clases UML](Manual_Java_UML) — Guía completa de Java 17+ y UML 2.5 con explicaciones detalladas sobre POO, clases, herencia, interfaces y relaciones UML.

## Ejercicios principales

- [01 - Figuras de Superhéroes](ejercicio01) — Práctica con clases básicas y composición de clases.
- [02 - Electrodomésticos](ejercicio02) — Práctica con clases básicas y herencia.
- [03 - Supermercado](ejercicio03) — Práctica con interfaces y polimorfismo.
- [04 - Servicios](ejercicio04) — Práctica con clases abstractas y jerarquías de herencia.
- [Ejercicios de POO](ejercicios_clases) — Colección de ejercicios variados de programación orientada a objetos.
- [Ejercicios resueltos UML](uml) — Ejercicios de POO modelados con diagramas UML (biblioteca, tienda, etc.).

## Ejercicios de oposiciones

- [Oposiciones Andalucía 2025](ejercicio_opos25AND) — Ejercicio de oposición: alumnos de ciclo formativo con módulos y calificaciones.
