
# HTML y CSS

![](images/lenguajes_1.png)

## Introducción

HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son los fundamentos de todo sitio web. HTML proporciona la estructura y el contenido, mientras que CSS se encarga del diseño visual y la presentación.

## Contenido del manual

1. **[Introducción a HTML](./introduccion_html)**
   Aprende los conceptos básicos de HTML: estructura de documentos, etiquetas, elementos de texto, listas, tablas, formularios y enlaces.

2. **[Introducción a CSS](./introduccion_css)**
   Descubre cómo dar estilo a tus páginas web con CSS: selectores, propiedades, modelo de cajas, Flexbox, diseño responsive y proyectos prácticos.

3. **[CSS Avanzado](./css)**
   Profundiza en CSS con selectores avanzados, pseudo-clases, posicionamiento, maquetación con floats y técnicas profesionales de diseño web.
